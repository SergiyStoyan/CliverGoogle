Cliver Google 4.*

Overview

CliverGoogle is a wrapper for Google.Apis

CliverGoogle's goals are:

- managing connection data;

- providing an easy API for operating objects;

Compatibility
CliverGoogle is compatible with .NET Standard 2.0.
